海报创意发散助手

围绕一个主题快速生成多套可执行的海报创意概念、构图和文案。

查看 SKILL.md GitHub 仓库

这是做什么的

帮助设计师从主题关键词中发散六个创意方向，覆盖直接表达、隐喻表达、反差表达、场景表达、符号表达和实验表达，并筛选推荐方案。

适合谁

视觉设计师、运营设计师、品牌设计师、设计学生

海报创意发散活动视觉主视觉社媒封面

典型场景

当你需要在短时间内为活动、课程、产品或节日主题发散海报创意时使用。

输入什么

海报主题、品牌/活动背景、目标受众、发布渠道、必须出现的信息、主标题或口号、设计尺寸、风格偏好、禁忌元素、截止时间

输出什么

主题拆解、六个创意方向、推荐方案、执行清单、变体建议

使用流程

  1. 1输入主题和渠道
  2. 2拆解关键词
  3. 3发散六个创意方向
  4. 4筛选推荐方案

示例

输入“AI 设计公开课招募”，输出六套海报概念、构图、标题和落地难度。
